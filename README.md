# Royal Barber Studio - Premium Barber & Hair Salon Website

A ultra-luxury, cinematic website for a modern barber and hair salon serving men, women, and kids. Built with React, Tailwind CSS, and Framer Motion for smooth animations and premium effects.

## Features
- **Ultra-Luxury Design**: Dark backgrounds with gold accents, glassmorphism, and glossy reflections.
- **Animations**: Fade-in on scroll, parallax hero, floating elements, hover scales, and gradient shines.
- **Responsive**: Mobile-first design with Tailwind CSS.
- **Sections**: Header, Hero, Services, About, Team, Pricing, Gallery, Booking, Footer.
- **Tech Stack**: React, Tailwind CSS, Framer Motion, React Scroll Parallax.

## Installation & Setup
1. Clone or download the project.
2. Run `npm install` to install dependencies (React, Tailwind, Framer Motion, etc.).
3. Configure Tailwind: Copy `tailwind.config.js` to your project root.
4. Add `index.css` to `src/` and import it in `index.js`.
5. Start the dev server: `npm start`.
6. Build for production: `npm run build`.

## Customization
- **Images**: Replace Unsplash URLs with your high-res photos.
- **Content**: Update text, services, and barbers in component files.
- **Booking**: Integrate with a backend (e.g., Node.js + MongoDB) for real form submissions.
- **Icons**: Use `react-icons` for custom icons (e.g., `MdContentCut` for scissors).
- **Colors**: Adjust in `tailwind.config.js` if needed.

## Performance Tips
- Use lazy loading for images (e.g., via `react-lazy-load-image-component`).
- Optimize animations for mobile devices.
- Test on various browsers and devices.
